Why to do this project?

Machine learning, as we know it today is being used for many different things, among one of them is weather data analysis and weather prediction. In this project we are going to develop a web application named Weather Prophecy. In this web application we will be developing a web application which gives user information about the weather of a particular Indian state or city. The main purpose of our application is to provide the user with extensive weather information, so that they can plan or schedule their day accordingly. This is also useful to people who just wants to check the weather. Weather Prophecy web application requires no user permissions for data, except their location. The location helps us to see which state the user is in and hence can show relevant data for weather in the city or state the user is in. This web application has mainly has two pages: Home and About. The user can directly search for a particular city in the search bar or can click on any of the states on the map of India. This will redirect the user to another page, where they will be given detailed information about previous as well as upcoming weather. Information like: temperature, pressure, wind speed and direction, etc will all be given in graphical format so that it is easy to understand.
